# Sudoku Solver

A Java-based Sudoku solver using a clean and efficient algorithmic approach.  
The solver can process any valid **9×9 Sudoku puzzle**, applying **backtracking with constraint validation** to find a solution.

---

##  Features

✔️ Solves 9×9 Sudoku puzzles  
✔️ Algorithmic approach using backtracking  
✔️ Clear and maintainable Java code  
✔️ Maven project structure for easy build & integration  
✔️ Can be used as a standalone solver or embedded in applications

---

##  Technologies

| Tech | Purpose |
|------|----------|

##  Getting Started

###  Clone the repository

```bash
git clone https://github.com/michaltomco/Sudoku-solver.git
cd Sudoku-solver
```

Build the project
```bash
mvn clean install
```

Run the solver

If a main method exists, run via:
```bash
java -jar target/Sudoku-solver-1.0.jar
```

Or run directly from your IDE (IntelliJ / VS Code / Eclipse).
